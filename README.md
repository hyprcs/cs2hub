# cs2hub

Public home for CS2 analytics tools, by **hyper**.

Every number on the site is generated by an automated pipeline and graded
against reality daily. Predictions are logged before matches start and the
record is published win or lose.

## Tools

| Path | Tool | What it is |
|---|---|---|
| `/veto/` | Veto Predictor | Predicts the full map veto (bans, picks, decider) for any CS2 matchup, with a public auto-graded track record |
| `/bets/` | Bet Ledger | Simulation-only paper bet ledger — flat 1-unit stakes, logged pre-match, per-bet closing-line value |
| `/record/` | Model Record | The prediction model's public scorecard — every logged pre-match probability, graded automatically |
| `/about/` | About | Case study: the pipeline, the models, and the honest-measurement rules |

## How it's built

Static site, no build step. Folders are URLs. Pages are regenerated by an
offline pipeline and pushed here; the host (Cloudflare Pages) serves the
files as-is. `_headers` carries the security and cache headers.

## Disclaimer

All betting content is a **simulation** — a paper record, no real money.
Nothing on the site is betting advice.
